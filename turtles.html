The Project

'Save the Turtles' is a website that two classmates and I created. We were asked to choose a cause that we cared about and work together to create a cohesive design.

The Problem

We learned how to navigate GitHub to build off each other's designs. We learned how to share our ideas and translate them into code. It was a great exercise in how to work remotely on a design and to match our code to our shared style guide.

My Role

The page I designed was "Why Turtles Matter". It provides information about sea turtles and the threats they face.
